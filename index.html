comandos basicos


git commit -m= guardar progreso

git log --oneline= revisar todos lospuntos de guardado actuales

git status -s= revisar que elementos se tienen almacenadps en git

git add = añadir un archivo a git

git reset --hard = restablecer un punto de guardado anterior

git add . = almacenar todos los archivos en el stage

git push = enviar las actualizaciones a github

git pull= descargar las actualizaciones de github a local

git clone(URL) = clonar y descargar la carpeta de github a local

git commit -am= añadir y guardar los cambios
